# electron-live2d-desktop
基于electron构建的看板娘PC端桌面挂件,项目资源属于本地加载。

### 概述
文件目录结构：

assets:live2d的相关文件。

Resources：看板娘的资源文件(包括模型资源)。

index.html：主页面。

main.js：electron入口文件。

package.json：electron的配置文件。


### 配置 & 运行预览

依赖包：

electron

electron-package

<br>
运行命令：

npm install cnpm -g --registry=http://registry.npm.taobao.org

cnpm install

cnpm start
